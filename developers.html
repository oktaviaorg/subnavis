The simplest way to access Bittensor data. Real-time prices, subnets analytics, wallet tracking, and whale alerts.

Introduction

The SubNavis API provides unified access to Bittensor network data, aggregated from multiple sources including TaoStats, CoinGecko, and the blockchain directly.

Base URL

https://subnavis.io/api/v1

Quick Start

# Get TAO price
curl https://subnavis.io/api/v1/price

# Get all subnets
curl https://subnavis.io/api/v1/subnets

# Get subnet details
curl https://subnavis.io/api/v1/subnet/1

Authentication

The free tier (Rebel) doesn't require authentication. For higher limits, include your API key in the header:

Authorization: Bearer YOUR_API_KEY

GET /subnets

GET /api/v1/subnets

Get all Bittensor subnets with computed metrics (APY, performance, emissions).

Query Parameters

sort Sort order: perf_7d (default), numeric, apy, emission
limit Number of results (default: 100)

Example

# Get subnets sorted by 7-day performance
curl "https://subnavis.io/api/v1/subnets?sort=perf_7d"

# Get top 10 by APY
curl "https://subnavis.io/api/v1/subnets?sort=apy&limit=10"

GET /subnet/:id

GET /api/v1/subnet/:id

Get detailed information about a specific subnet, including top validators.

Path Parameters

id Subnet ID (netuid), e.g., 1, 8, 19

Example

# Get Subnet 1 (Prompting)
curl "https://subnavis.io/api/v1/subnet/1"

# Get Subnet 8 (Taoshi)
curl "https://subnavis.io/api/v1/subnet/8"

Error Handling

The API uses standard HTTP status codes and returns errors in a consistent format:

{
  "success": false,
  "error": "Error type",
  "message": "Human-readable description"
}
200Success
400Bad Request - Invalid parameters
401Unauthorized - Invalid or missing API key
404Not Found - Resource doesn't exist
429Too Many Requests - Rate limit exceeded
500Server Error - Something went wrong

Rate Limits

Rate limits are applied per API key (or IP for unauthenticated requests):

  • Rebel (Free): 500 requests/day
  • Builder: 10,000 requests/day
  • Degen: 50,000 requests/day
  • Whale: Unlimited

Rate limit headers are included in responses:

X-RateLimit-Limit: 500
X-RateLimit-Remaining: 423
X-RateLimit-Reset: 1706998800
